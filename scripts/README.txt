SCRIPTS DIRECTORY
=================

Utility scripts for installation, verification, and system checking.

Files:
------
- check_system.py          Check system capabilities and get recommendations
- health_check.py          Comprehensive codebase health check
- verify_installation.py   Verify that installation is working correctly

Usage:
------
python scripts/check_system.py
python scripts/health_check.py --skip-deps
python scripts/verify_installation.py

Note: Run from project root directory
